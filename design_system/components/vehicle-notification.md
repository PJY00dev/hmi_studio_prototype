# Vehicle Notification (Driving Notification)

## 소스
- 웹: `https://document.pleos.ai/docs/connect/guide/docs-design/components/notification/vehicle-notification`
- Figma 노드: 미정의

---

## 구조 (Anatomy)

| 요소 | 설명 |
|------|------|
| Root | 전체 컨테이너 |
| Telltale icon | 차량 상태 아이콘 |
| Title | 알림의 주요 메시지 |
| Description | 부가 설명 텍스트 |
| Guide image | 가이드 이미지 (선택적) |

---

## 사용 규칙
- 주행 중 차량 상태 변화 알림에 사용
- 운전자 시야 방해 최소화
- 짧고 명확한 메시지만 사용
- 광고성 알림 불가 (운전자 관련 정보만 허용)
